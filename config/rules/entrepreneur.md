# Entrepreneur & Business Rules

## Product Development
- Focus on MVP first, iterate based on feedback
- Validate assumptions before building features
- Prioritize user experience and simplicity
- Document decisions and rationale

## Business Analysis
- Use CEO/CTO advisor skills for strategic decisions
- Leverage market-researcher and competitive-analyst skills
- Track metrics that matter (not vanity metrics)

## Project Management
- Use Linear/Jira integration skills for task tracking
- Break work into small, deliverable chunks
- Regular progress updates and retrospectives

## Marketing & Content
- Use content-marketer and SEO skills
- Focus on value-driven content
- Build in public when appropriate
